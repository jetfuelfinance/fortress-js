    example

(async function() {
  const castVoteTx = await fortress.castVoteBySig(

      r: '0x130dbcd2faca07424c033b4479687cc1deeb65f08509e3ab397988cc4c6f2e78',
      s: '0x1debcb8250262f23906b1177161f0c7c9aa3641e6bff5b6f5c88a6bb78d5d8cd'
    }
  );
  console.log('Ethers.js transaction object', castVoteTx);
})().catch(console.error);

    • support: boolean

      A boolean of true for 'yes' or false for 'no' on the proposal vote.

    • signature: Signature

      An object that contains the v, r, and, s values of an EIP-712 signature.

createVoteSignature

  • createVoteSignature(proposalId: number, support: boolean): Promise<Signature>
    • Defined in gov.ts:172

    Create a vote signature for a Fortress Governance proposal using EIP-712. This can be used to create an 'empty ballot' without burning gas. The signature can then be sent to someone else to post to the blockchain. The recipient can post one signature using the castVoteBySig method.

    example
    const fortress = new Fortress(window.ethereum);

(async () => {

  const voteForSignature = await fortress.createVoteSignature(20, true);
  console.log('voteForSignature', voteForSignature);

  const voteAgainstSignature = await fortress.createVoteSignature(20, false);
  console.log('voteAgainstSignature', voteAgainstSignature);

})().catch(console.error);

    Parameters

    • proposalId: number

      The ID of the proposal to vote on. This is an auto-incrementing integer in the Governor Alpha contract.

    • support: boolean

      A boolean of true for 'yes' or false for 'no' on the proposal vote. To create an 'empty ballot' call this method twice using true and then false for this parameter.

    Returns Promise<Signature>

    Returns an object that contains the v, r, and s components of an Ethereum signature as hexadecimal strings.
